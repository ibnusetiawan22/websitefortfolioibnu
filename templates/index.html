Muhammad Ibnu Setiawan

Junior Developer | Network Administrator

Saya adalah mahasiswa D3 Teknologi Komputer Universitas Bina Sarana Informatika semester 4. Saya memiliki pengalaman magang 6 bulan di Aldi Games Playstation, di mana saya mengelola server game, membantu pemain, dan menyelesaikan masalah teknis. Saya yakin dengan kombinasi pendidikan formal, pengalaman magang, dan program MSIB, saya akan siap untuk memasuki dunia kerja dan memberikan kontribusi yang positif di bidang teknologi.
